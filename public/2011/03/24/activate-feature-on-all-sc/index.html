Activate Feature on all Site Collections in a Web Application

This is pretty easy using powerhell with SharePoint 2010:

[powershell]Get-SPWebApplication "http://WebApplicationUrl:PortIfNeeded" | Get-SPSite | ForEach-Object{Enable-SPFeature "Solution.NameFeatureName" -url $.Url}[/powershell]
