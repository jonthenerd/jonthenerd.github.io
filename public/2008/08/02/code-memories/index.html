Code Memories?

In fact - many parts of my C/C++ education were /very/ lacking. However, there are times when I think back to things that I could have done better, things I really enjoyed coding (Blackjack console app with ASCII cards..heh), and things I just plain didn’t understand or care to.Quick Google search brought up http://www.cplusplus.com/doc/tutorial/ for relearning some of that forgotten knowledge.
